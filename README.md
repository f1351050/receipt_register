# レシート管理アプリ
レシート画像から必要な情報を読み取り、表示・登録ができます。

# DEMO
![レシート管理（ポートフォリオ）](https://github.com/f1351050/receipt_register/assets/126868552/1620758f-a57c-4431-9416-9d7e89e43da0)

# 使用技術  
-python3 
# 環境：Requirement 
- Python 3.7.3
- conda 4.7.12 

# 機能一覧
-レシートの読み取り  
-カテゴリー登録  
-購入履歴の表示  
-読み取った情報から小計を表示
